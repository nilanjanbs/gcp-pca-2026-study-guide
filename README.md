## 🎯 Overview
This repository contains a complete, cohort-style study framework for the **Google Professional Cloud Architect (PCA) 2026** certification. It aligns with the official v6.1 exam guide, includes deep-dive domain guides, scenario-based practice, curated resources, and a structured 8-week learning plan. Designed for self-study or collaborative cohort prep.
